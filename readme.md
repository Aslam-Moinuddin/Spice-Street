# Spice Street

## Overview
Spice Street is a full-stack restaurant application built with the MERN stack (MongoDB, Express, React, Node.js). The application allows users to browse the restaurant menu, place orders, and manage reservations seamlessly. This project showcases a modern web application architecture and provides a user-friendly interface for a delightful dining experience.

## Features
- **User Authentication:** Secure login and registration for users.
- **Menu Browsing:** View a list of available dishes with detailed descriptions and prices.
- **Order Placement:** Users can place orders for delivery or pick-up.
- **Reservation Management:** Users can reserve tables at the restaurant.
- **Admin Dashboard:** Admins can manage menu items, view orders, and handle reservations.
- **Responsive Design:** The application is fully responsive and works on various devices.

## Tech Stack
- **Frontend:** React, Redux (for state management), Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aslam-Moinuddin/Spice-Street.git
   cd spice-street
2. **Set up the backend:**
    Navigate to the backend directory:
    ```bash
    cd backend
3. **Install dependencies:**
    ```bash
    npm install
4. **Create a .env file in the backend directory and add the following environment variables:**
    ```bash
    DB_URI=mongodb+srv://your-mongo-uri
    JWT_SECRET=your-jwt-secret
    PORT=4000
5. **Start the server:**
    ```bash
    npm run dev
    
6. **Set up the frontend:** 
    **Navigate to the frontend directory:**
    ```bash
    cd ../frontend

7. **Install dependencies:**
    ```bash
    npm install
8. **Start the React app:**
    ```bash
    npm run dev
